# student-blog-webpage
A one-page personal blog developed using HTML and CSS for the BBT2102 Web Application Development Individual Coding Challenge.
# A Day in My Life at University

## Overview

The website presents an overview of my university life as a Diploma in Business Information Technology (DBIT) student at Strathmore University.

## Features

- Navigation menu with internal links
- Personal introduction
- Description of a typical university day
- Favourite place on campus
- Advice for new students
- Reflection on my university experience
- Personal information footer
- CSS styling and hover effects

## Technologies Used

- HTML5
- CSS3
